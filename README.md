# **Portfolio Website**

## **Overview**
This is my personal portfolio website showcasing my projects, skills, and achievements.  
It is built using **HTML, CSS, and JavaScript** and is fully responsive for desktop and mobile devices.

## **Features**
- Responsive design for **mobile and desktop**  
- Projects showcase with links and descriptions  
- Skills section with proficiency indicators  
- Contact form for easy communication

## **Technologies Used**

- **CSS3** – Styling, layouts, and animations  
- **JavaScript** – Interactive functionality
